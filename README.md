# 📅 Daily Code

A personal repository for daily coding practice — algorithms, data structures, problem-solving, and mini-projects, one day at a time.

---

## 🎯 Purpose

The goal of this repo is to build consistency and sharpen problem-solving skills through daily practice. Each entry captures a coding challenge, solution, or concept explored that day.

---

## 🗂️ Topics Covered

- Arrays & Strings
- Linked Lists
- Stacks & Queues
- Trees & Graphs
- Recursion & Backtracking
- Dynamic Programming
- Sorting & Searching
- Hash Maps & Sets
- Math & Bit Manipulation
- System Design concepts

---

## 🚀 Getting Started

Clone the repo and explore any day's folder:

```bash
git clone https://github.com/GdAyo19/daily-code.git
cd daily-code
```

Each day's folder contains a `problem.md` describing the challenge and a solution file in the language of choice.

## 📌 Rules

1. Code every day — no skipping.
2. Understand the solution before committing it.
3. Add a brief explanation or comment in each solution file.
4. Revisit hard problems after a few days.
